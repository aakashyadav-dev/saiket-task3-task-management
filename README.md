# TaskFlow - Task Management Application

A modern, production-ready task management application built with React.js, featuring a beautiful glassmorphism UI and complete task management capabilities.

## 🚀 Features

### Core Features
- ✅ Add, Edit, and Delete Tasks
- ✅ Mark Tasks as Complete/Incomplete
- ✅ Task Statistics (Total, Completed, Pending, Completion Rate)
- ✅ Local Storage Persistence
- ✅ Responsive Design

### Advanced Features
- 🔍 Search Tasks
- 🎯 Filter by All/Active/Completed
- 📊 Sort by Newest/Oldest/Priority
- 📅 Due Date Management
- 🏷️ Task Priority (High/Medium/Low)
- 🎨 Dark Mode Toggle
- 💬 Toast Notifications
- 🔔 Confirmation Dialogs
- 📱 Mobile-First Design

## 🛠️ Technologies Used

- **React 18** - UI Library
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **React Icons** - Icons
- **UUID** - Unique ID Generation
- **Local Storage** - Data Persistence

